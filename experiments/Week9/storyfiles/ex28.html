Experiment : Artsy fetch data

Experiment

Type a text related to art /art history and click on search button.


Description

In this experiment I am using artsy API . Here I making a json request to search artist, artwork or gene from Artsy api.
Here it search for anything by following the search link from root with a query parameter.

curl -v "https://api.artsy.net/api/search?q=Andy+Warhol" -H "X-XAPP-Token:..."

The response is a paginated result with embedded search results.
